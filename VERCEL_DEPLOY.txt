🚀 VERCEL DEPLOYMENT - QUICK STEPS

═══════════════════════════════════════════════════════════════

STEP 1: Deploy Backend
═══════════════════════════════════════════════════════════════

1. Go to: https://vercel.com/new
2. Import: github.com/Raghahahav/ILikeITBlack
3. Configure:
   - Project Name: nova-backend
   - Framework: Other
   - Root Directory: backend
   - Build Command: (leave empty)

4. Add Environment Variables:
   OPENROUTER_API_KEY = sk-or-v1-your-key-here
   MODEL_NAME = anthropic/claude-3-haiku
   CORS_ORIGINS = *

5. Deploy and copy URL (e.g., https://nova-backend-xxx.vercel.app)

═══════════════════════════════════════════════════════════════

STEP 2: Deploy Frontend
═══════════════════════════════════════════════════════════════

1. Go to: https://vercel.com/new (import same repo again)
2. Configure:
   - Project Name: nova-frontend
   - Framework: Vite
   - Root Directory: frontend
   - Build Command: npm run build

3. Add Environment Variable:
   VITE_API_URL = [YOUR BACKEND URL FROM STEP 1]

4. Deploy and visit your site!

═══════════════════════════════════════════════════════════════

⚠️  IMPORTANT:
═══════════════════════════════════════════════════════════════
